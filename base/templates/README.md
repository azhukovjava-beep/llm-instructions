# Templates

`templates/` хранит шаблоны сборки платформенных артефактов из нормализованной базы.

Примеры:

- шаблон генерации `AGENTS.md`;
- шаблон генерации `CLAUDE.md`;
- шаблоны rule-файлов для Cursor;
- шаблоны summary / report для sync-агента.

## Правило

В шаблонах не должно быть бизнес-смысла как единственного источника правды. Смысл живет в `schema/`, а здесь лежит только форма представления.
