# Docs Sources

`docs-sources/` хранит ссылки на официальную документацию провайдеров и краткие заметки по ней.

Здесь не надо копировать документацию целиком. Нужно хранить:

- официальный URL;
- что именно проверять во время sync;
- дату последней ручной или автоматической сверки;
- важные замечания для маппинга.
