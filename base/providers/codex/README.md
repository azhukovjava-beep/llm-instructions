# Codex Provider Model

Здесь описывается, как сущности общей базы выражаются в Codex.

Примеры того, что здесь стоит фиксировать:

- `AGENTS.md` как основной файл инструкций;
- `.codex/skills` и связанные расширения;
- правила области действия инструкций;
- ограничения и формат, которые нужно учитывать при генерации из общей модели.
