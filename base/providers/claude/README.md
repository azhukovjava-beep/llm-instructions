# Claude Provider Model

Здесь описывается, как сущности общей базы выражаются в Claude Code.

Примеры того, что здесь стоит фиксировать:

- `CLAUDE.md` как основной memory / instruction файл;
- `.claude/`-структуры;
- rules, skills, plugins, commands, hooks;
- ограничения на формат и размещение;
- что можно перенести из `schema/` напрямую, а что требует отдельной адаптации.
