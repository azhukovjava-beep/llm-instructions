# Cursor Provider Model

Здесь описывается, как сущности общей базы выражаются в Cursor.

Примеры того, что здесь стоит фиксировать:

- `.cursor/rules` и связанные rule-файлы;
- поддержка `AGENTS.md` как дополнительного или упрощенного источника;
- особенности scope, вложенности и project rules;
- различия между тем, что можно держать в общей модели, и тем, что лучше оставлять Cursor-specific.
