# Agent Packaging

Здесь нужно описывать, как один и тот же нормализованный агент упаковывается для разных платформ.

Примеры:

- Claude: `.claude/agents/<name>.md`
- Codex: portable default через skill, optional native multi-agent setup
- Cursor: portable default через requested/manual rule

Это помогает не терять общий смысл при разном runtime format.
