# Agent Profiles

Здесь должны лежать нормализованные профили агентов.

Примеры:

- `code-reviewer`
- `react-specialist`
- `database-administrator`
- `security-engineer`
- `release-manager`

Для каждого профиля полезно указывать:

- цель;
- триггеры использования;
- ограничения;
- требуемые навыки;
- как этот профиль выражается в Claude, Codex и Cursor.
